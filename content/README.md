# content/

Put your own content packs here, then point `config.js` at them (or use a URL):

- **quotes** — `local:content/quotes.js` sets `window.CONTENT_QUOTES = [ ["quote", "Author"], … ]`
- **poems**  — `local:content/poems.js`  sets `window.CONTENT_POEMS  = [ ["original or ''", "body\nlines", "Author"], … ]`
- **decks**  — `local:content/decks.js`  sets `window.CONTENT_DECKS  = [ { name, mode, cards }, … ]`

Nothing ships here by default, on purpose — it keeps the public repo free of
copyrighted or personal content. **Only add material you have the right to publish**
(public-domain, your own, or explicitly licensed).
